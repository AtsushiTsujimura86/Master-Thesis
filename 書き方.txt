\chapter{背景技術} 
本開発では、通勤・住宅届を申請するためのwebページ上のフォームを作成した。以下ではフォームを作成するために必要な技術とその概要を示す。
\section{webページの実装}
\subsection{HTML}
\begin{par}
  HTMLとはHyper Text Markup Languageの略であり、webサイトのコンテンツの構造を作るためのマークアップ言語である。
\end{par}
\begin{par}
  HTMLではタグという部品を組み合わせてコンテンツを作っている。タグの例として、webページの見出しに使われるh1タグや、ユーザからの入力を受け付けるinputタグ、表を作る際に使われるtableタグなどがある。
\end{par}
\begin{par}
  タグにはクラスやidを指定することができる。複数のタグにある一つのクラスを指定することはできるが、あるidはHTML中にに一つしか存在してはいけない。クラスやidは後述するCSSやJavaScriptにおいて、特定のクラスやあるidにのみ処理を施したいという場合に必要である。
\end{par}
\begin{par}
  HTMLのコードは大きく分けて、HTMLのメタ情報や外部リソースへのリンクなどを記述するhead要素と実際にwebページに表示する内容を記述するbody要素からなる。実際のHTMLのソースコードの例をソースコード\ref{html_src_1}に示す。\\
\end{par}



\begin{lstlisting}[caption=HTMLのソースコードの例,label=html_src_1]
  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="UTF-8">
      <title>ページタイトル</title>
    </head>
    <body>
      <h1>h1タグのサンプル</h1>
      <input type="text" id="input_1">
    </body>
  </html>

  \end{lstlisting}

\chapter{システムの説明}

\section{システムの概要}
\begin{par}
今回開発したシステムは、大学職員が提出する通勤・住宅届の申請をオンラインで行えるようにするシステムである。システムの概要図を図\ref{fig:system_img_1}に示す。ユーザ（職員）は通勤・住宅届の申請フォームをWeb上で開き、申請内容（氏名や職員番号、届出の理由、通勤経路、家賃額など）入力し、必要に応じて証明書類の画像ファイルを添付する。ユーザが提出すると、Google Maps APIにより通勤距離の計算を行い、その後GASにより以下のような処理が行われる。

\begin{itemize}
  \item 申請内容を抽出
  \item 手当額の計算等の処理
  \item Googleドライブ上に職員ごとのフォルダ（個人用フォルダ）を作成し、添付された画像ファイルをアップロード
  \item 個人用フォルダに、Googleスプレッドシート（個人用スプレッドシート）を作成し、申請内容を入力
  \item 申請をしたすべての職員の申請内容を蓄積するためのスプレッドシートに申請内容を入力
  \item 担当部署1宛てに申請内容を記載したメールを送信
  \item 担当部署1の職員が受信したメールを開き、申請内容を確認したうえでメール内に設置してあるボタンを押下すると担当部署2宛てに同様のメールを送信
\end{itemize}
